2. Pupil-Teacher Ratio

There are 5000+ data sets available on data.gov.sg.

In this exercise, practice querying and visualizing data on the Pupil-Teacher Ratio (PTR) in primary schools by year. Try reproducing the plot below.

https://data.gov.sg/datasets/d_bc5a1a10afedf7d25128dae1342c9556/view

In[43]:
# Type your code here:
import requests

dataset_id = "d_bc5a1a10afedf7d25128dae1342c9556"
url = "https://data.gov.sg/api/action/datastore_search?resource_id="  + dataset_id
        
response = requests.get(url)
response_json = response.json()
records = response_json["result"]["records"]
df = pd.DataFrame(records)

df['pupil_teacher_ratio'] = pd.to_numeric(df['pri_students_to_teachers'])
df['year'] = pd.to_numeric(df['year'])

df.plot(kind = 'line', x = 'year', y = 'pupil_teacher_ratio', 
         linewidth = 1, color = 'purple', legend = False,
         title = 'Pupils Per Teacher in Primary Schools (1981 to 2024)', 
         xlabel = 'Year', ylabel = 'Pupil-Teacher Ratio')
tick_values = list(range(1980, 2030, 5))
plt.xticks(ticks = tick_values, labels = tick_values, rotation = 45)
plt.grid(axis='y', linestyle = ':')
plt.show()
